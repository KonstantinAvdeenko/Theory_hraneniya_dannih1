 Была спроектированна логическая структура БД для создания библиотеки, которая содержит следующие таблицы:
Читатели (Номер читательского билета, ФИО, Адрес,  Телефон)
Книги (Шифр книги, Издательство, Название, Автор(ы), Год издания, Объем (в страницах), Цена, Количество экземпляров в фонде)
Издательства (id, Название, Город)
Выдача книг (id, Название, Наличие, Шифр книги, Кому была выдана книга)

 Для библитеки самым важным является учет книг, поэтому логическая структура спроектирована для учета книг.

В таблицу "Книги" было добавлено числовое поле "Издательство" для связи с таблицей "Издательство", где первичный
ключ поле с уникальным "id" каждого издательства, которое добавляется в поле "Издательство" таблицы "Книги". 
Это сделано из соображения того,что у издательства может быть много книг, а у каждой книги только одно 
издательство, которое выбирается из таблицы "Издательство".

 Так, как у каждой книги может быть несколько экземпляров в фонде библиотеки, а читатель может брать много книг,
то между таблицами "Книги" и "Читатель" - была связь многие ко многим, что крайне не желательно. Так, как 
библиотеку в первую очередь интересуют книги, то была добавлена таблица "Выдача книг", где одна определенная 
книга имеющая несколько экземпляров - может выдаваться много раз, а экземпляры книги могут при этом выдаваться 
разным читателям. Связь таблицы "Книги" и "Выдача книг" имеет отношение 1 ко многим и соединяется по ключевому 
полю "Шифр книги" таблицы "Книги" с полем вторичного внешнего ключа "Шифр книги" таблицы "Выдача Книг". 
Каждая выдача книги читателю - должна иметь свой уникальный "id" номер со своим полем. Также для отслеживания 
наличия книг было добавлено булево поле "Наличие" в таблице "Выдача книг".

 Так, как 1 читатель берет 1 экземпляр каждой книги, но при этом может взять много разных книг, то связь между
таблицей "Выдача книг" и "Читатель" имеет отношение много к 1 так, как учет книг учитывается в разрезе каждого
определенного читателя. Связываются таблицы по вторичному внешнему ключу поля "Кому была выдана книга" из 
таблицы "Выдача книг" с первичным ключом поля "Номер читательского билета" из таблицы "Читатели".
